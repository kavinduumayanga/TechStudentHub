JavaScript Basics for Absolute Beginners

JavaScript is the language that makes web pages interactive. You can use it to respond to clicks, validate forms, filter lists, and update text without reloading the page. This guide teaches the fundamentals you need to write small, reliable scripts—and understand what the browser is doing.

What JavaScript is used for

On a static site, JavaScript is usually used for:

  • Filtering and searching content (like a blog search)
  • Opening menus and toggles
  • Form validation (checking required fields)
  • Small UI improvements (tabs, accordions)

This site’s blog search is powered by a simple script. You can see it in action on Blog.

Variables and types (just the essentials)

Variables store values. Use const when the variable reference won’t change, and let when it will.

const siteName = "Tech & Student Help Hub";
let count = 0;
count = count + 1;

Common data types you’ll use early:

  • String: text like "Hello"
  • Number: 42, 3.14
  • Boolean: true, false
  • Array: a list of values
  • Object: a set of named properties

Conditions and loops

Conditions let you run code only when something is true:

const age = 18;
if (age >= 18) {
  console.log("Allowed");
} else {
  console.log("Not allowed");
}

Loops repeat work. A simple for loop is common:

const items = ["HTML", "CSS", "JavaScript"]; 
for (let i = 0; i < items.length; i++) {
  console.log(items[i]);
}

Later you’ll use methods like forEach and map, but start simple.

Functions (your most important tool)

Functions group logic and help you reuse code:

function greet(name) {
  return `Hello, ${name}!`;
}

const message = greet("Student");
console.log(message);

Beginner habit: write small functions that do one thing. It makes debugging easier.

Arrays and objects (real-world data)

Arrays store lists. Objects store “things” with properties.

const posts = [
  { title: "HTML Basics", minutes: 11 },
  { title: "CSS Fundamentals", minutes: 12 }
];

const firstTitle = posts[0].title;

Most real code is simply “get data → loop → render”.

DOM basics: changing the page

The DOM (Document Object Model) is how JavaScript interacts with HTML. You can select elements and update them.

const heading = document.querySelector("h1");
heading.textContent = "Updated title";

Two common properties:

  • textContent for plain text
  • innerHTML for HTML (use carefully)

Events: clicks, input, submit

Events trigger code when the user does something:

const button = document.querySelector(".button");
button.addEventListener("click", () => {
  console.log("Clicked!");
});

Input events help you build search/filter features:

const input = document.querySelector("#blog-search");
input.addEventListener("input", () => {
  console.log(input.value);
});

Practice exercises

Practice is where JavaScript becomes real. Try these beginner exercises:

  1. Counter: a button that increments a number on the page
  2. Checklist: mark items complete with a click
  3. Filter: hide/show cards based on a search input
